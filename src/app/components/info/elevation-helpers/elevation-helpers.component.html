Applying Elevation

The Material Design specification gives guidance on expressing elevation on UI elements by adding shadows. Angular Material provides CSS classes and Sass mixins for adding these shadows.

Elevation CSS classes

The core-theme Sass mixin, described in the theming guide, emits CSS classes for applying elevation. These classes follow the pattern mat-elevation-z#, where # is the elevation number you want, from 0 to 24. These predefined classes use the CSS box-shadow settings defined by the Material Design specification.You can dynamically change elevation on an element by swapping elevation CSS classes.
